                                     Git 合并远程分支


   有的时候我们需要跟别人合作进行开发，然后分别使用不同的Git分支，等项目完成时，
需要进行代码合并，就需要知道Git如何合并远程分支。
步骤

   第一步 切换到分支b

   git checkout b

   该指令的意思：创建一个本地分支，并将远程分支放到该分支里面去。

   第二步 将远程代码pull到本地

   git pull origin b

   第三步 返回到你的分支a

   git checkout a

   第四步 合并分支a与分支b

   git merge b
   
   该指令的意思：当前所在分支与b进行合并。

   第五步 把本地的分支a同步到远程

   git push origin a
